## What This Is
A dual-brand website for a wellness studio (AZAI) and specialty coffee bar (BRUME) that share the same physical space in Jal El Dib. Both brands have dedicated pages under one unified site.

## Tech Stack
- Next.js 15.3 App Router + TypeScript
- Tailwind CSS v4 with custom brand tokens
- Framer Motion for all animations
- Deployed on Vercel (domain TBD)

## Design Rules for This Project
- No emojis anywhere in UI, copy, or code. They are cheap and off-brand.
- No em dashes (--) in any content, copy, or UI text. Use a regular hyphen or restructure the sentence.
- Cinematic and moody — dark backgrounds, serif italic headlines, spring animations
- AZAI = dark/athletic/terracotta. BRUME = warm/cafe/golden
- Never break the dual-brand consistency — both sides always feel like one universe
- Framer Motion on everything — no static sections
- All Book buttons go to WhatsApp: https://wa.me/96171607705
- English only for now
